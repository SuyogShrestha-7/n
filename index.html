About

Hi, I'm Suyog Shrestha a passionate full-stack developer, cybersecurity enthusiast, and BSc Computing student from Nepal. I specialize in building modern, responsive web applications and enjoy exploring the intersection of technology, machine learning, and cybersecurity. Whether it's developing smart applications with Java and Spring Boot, crafting clean frontends with HTML, CSS, and JavaScript, or diving into systems and networking, I'm constantly learning, experimenting, and evolving as a developer.

Driven by curiosity and a desire to solve real-world problems through clean, efficient code, I strive to build meaningful digital experiences. Outside of tech, you’ll find me immersed in music, playing chess, or working on my fitness goals.

Let’s connect and create something impactful together.

What I do

Front-End Development

I build responsive, user-friendly web interfaces using HTML5, CSS3, Bootstrap, and JavaScript. My focus is on clean UI design and accessibility, ensuring websites work across all devices.

Back-End Development

I develop scalable backend systems using Java with Spring Boot, PHP, and MySQL. From REST APIs to full-stack integration, I enjoy solving logical problems and building secure systems.

Cybersecurity & Systems

I'm passionate about system-level security, Linux, networking, and ethical hacking. As a BSc Computing student, I'm diving deep into C programming, cybersecurity fundamentals, and digital safety.
